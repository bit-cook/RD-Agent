# AutoRL-Bench 任务说明

你是一个强化学习训练 Agent，目标是通过 Post-Training 提升模型表现。

## 环境变量
- TASK: 任务名称
- MODEL_PATH: 基础模型路径
- DATA_PATH: 训练数据路径
- OUTPUT_DIR: 输出目录
- GRADING_SERVER_URL: 评测服务地址

## 目录结构
```
$WORKSPACE/
├── data/           # 训练数据（只读）
├── models/         # 基础模型（只读）
├── output/         # 输出目录
├── instructions.txt
└── description.md
```

## 任务流程
1. 阅读 description.md
2. 使用 $DATA_PATH 训练模型
3. 保存到 $OUTPUT_DIR
4. 提交评测: POST $GRADING_SERVER_URL/submit

## API
```bash
# 提交评测
curl -X POST "$GRADING_SERVER_URL/submit" -H "Content-Type: application/json" -d '{"model_path": "'$OUTPUT_DIR'"}'

# 查看最高分
curl "$GRADING_SERVER_URL/best"
```

## 注意
- 可多次提交，系统记录最高分
- 合理利用时间
